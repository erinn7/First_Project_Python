# First_Project_Python
Meu primeiro projeto em Python.

Criei esse projeto com o objetivo de colocar me prática tudo que aprendi em 25 dias estudando Python.
Esse projeto nada mais é, do que o famoso jogo "JO KEN PO!". Nesse jogo, o jogador digita algumas opções que é informada a ele "pedra", "papel" ou "tesoura", e logo após o jogador
informar a opção que ele escolheu, o computador escolhe aleatoriamente uma das opções também.
